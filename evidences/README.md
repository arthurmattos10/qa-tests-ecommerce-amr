# Evidências de Testes

Esta pasta contém as evidências coletadas durante a execução dos testes do projeto **E-commerce AMR**.

As evidências incluem:

- Capturas de tela (screenshots)
- Registros visuais de execução dos testes
- Comprovação dos resultados obtidos

O objetivo é documentar o comportamento do sistema durante os testes funcionais, exploratórios e de responsividade.
